# AI-Powered-Amazon-S3-Documentation-Assistant-using-RAG
AI-powered Amazon S3 Documentation Assistant using RAG, NLP, and Semantic Search. Retrieves relevant answers from Amazon S3 documentation stored in AWS S3 using Python, Boto3, PyPDF, and Sentence Transformers.
Features
Download documentation from AWS S3
Extract text from PDF documents
Intelligent document chunking
Semantic search using NLP
Natural language question answering
Retrieval-Augmented Generation (RAG) workflow
AWS integration using Boto3
Technologies Used
Python
AWS S3
Boto3
PyPDF
Sentence Transformers
Scikit-Learn
NumPy
NLP
RAG (Retrieval-Augmented Generation)
